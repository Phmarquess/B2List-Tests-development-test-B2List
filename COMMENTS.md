#Decisão da arquitetura utilizada

 Nesta apricacao foi desenvolvida usando Reactjs foi utilizado a biblioteca Formik a ajuda a facilitar na criação de criação de formulários, React router para criações de rotas , Axios para fazer requisições, Yup para fazer validações em nossos formulários, React Dom  para renderiza o elemento React no navegador o banco de dados Mysql como solicitado , Express para gerenciar requisições de diferentes HTTP em diferentes URLs e também foi utilizado o cors
